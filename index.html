How to Use

  1. Access Generator: Click the button below to go to the token generator
  2. Select Range: Choose the token range you want to generate
  3. Customize Label: Change the meal type label (applies to all tokens)
  4. Edit Tokens: Click Edit Tokens to modify individual numbers
  5. Print: Print tokens directly from your browser
  6. Distribute: Cut and distribute tokens to students
